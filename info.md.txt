Smart Library Management System with the following approach:

### **Technical Architecture:**

- **Frontend:** React Native (Expo) + TypeScript + Expo Router

- **Backend:** Node.js (API Gateway) + Laravel (Core APIs)

- **Database:** MySQL

- **Auth:** JWT Tokens

- **QR Codes:** Standard QR with JSON data

- **State:** Zustand + React Query for caching

- **UI:** React Native Paper + Custom Components

- **Scanner:** Expo Camera + Barcode Scanner

### **All Features (No Skipping):**

✅ Student Registration with PENDING status

✅ Login with JWT

✅ Student Dashboard

✅ Digital Library Card (QR + Barcode)

✅ Book Browsing (Search, Filter, Details)

✅ QR/Barcode Scanning

✅ Loan Management (Active, History, Overdue)

✅ Academic Calendar

✅ Profile & Settings

✅ Offline Mode with MMKV + React Query
