1. INTRODUCTION
The Mobile App is the student-facing application of the Smart Library System.
It enables students to:
•	Register using college details
•	Wait for admin approval
•	Login using ID number
•	View digital library card (QR + barcode)
•	Check issued books, due dates, overdue alerts
•	Browse/search available library books
•	Scan book barcodes to view book details
•	View academic calendar
•	Manage profile & settings
•	Receive push notifications for due reminders
This application provides a smooth, fast, and secure user experience.
________________________________________
2. OBJECTIVES
Primary Objectives
•	Provide seamless mobile access for library users.
•	Digitize the library card with barcode & QR.
•	Enable quick book lookups via scanning.
•	Improve communication using push notifications.
Secondary Objectives
•	Improve student experience.
•	Reduce dependency on physical cards.
•	Increase library transparency and automation.
